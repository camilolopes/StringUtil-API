StringUtil-API
It is a simple API to remove character duplicated in sequence from string. 

How to use? 
Only add dist-jar/StringUtil.jar in our project. 

Can I add new functionally? 
Yes, you can. Make checkout project and add new functionally to project. Let me know of your update, please. 

Why structure of this project is very simple?
